# Birth Year Calculator
This system asks the user for their age and calculates the year they were born.
It uses try/except to handle invalid input safely.

## Features
- Runs until the user types "stop"
- Converts input to an integer
- Calculates birth year using 2026 - age
- Catches invalid input (ValueError)
- Catches datatype errors (TypeError)

# How to use
Type your age to see your birth year.
Invalid input gives an error message.
Type "stop" to exit.


## Learning Purpose
Practice with:
- exception handling
- basic arithmetic
- loops and input validation
